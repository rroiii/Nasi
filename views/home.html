What is Our Mission?


The correlation between population growth, agricultural land reduction, and food security is a complex issue with various interconnected factors. The challenge of providing sustenance for a continually increasing population is a significant problem. So, what are the problems that need to be addressed?

1. How can the land cover model in West Java for the years 2020 and 2032 be developed using Artificial Neural Network (ANN) in deep learning?

2. How can the population growth model in West Java for the year 2032 be constructed using Artificial Neural Network (ANN) in deep learning?

3. How can the future food security predictions be made based on models incorporating population growth and agricultural land expansion?

From these issues, four models will be developed, namely the land cover model in West Java for the years 2020 and 2032, as well as the population growth model in West Java for the year 2032, using deep learning Artificial Neural Network (ANN).
